EPICA PROYECTO FINAL INTEGRADOR  MODULO 1 (09/2023)
OBJETIVO: Creación de un Foro Personalizado para Publicar Contenido Multitemático

Qué hace el proyecto?
    crear y compartir publicaciones de temas variados en una página web personal. 
    Cada publicación debe incluir un título descriptivo, contenido informativo, una fecha de creación y la opción de 
    adjuntar el link de una imagen de internet para relacionarla con el contenido. 
    En esta etapa del trabajo, no se requiere manejo de comentarios para las publicaciones.

Que tecnologias se aplican?
    Tecnologías de programación web, como Node.js, EJS, MySQL y Sequelize, Bootstrap,

Que actividades de programacion se prentenden reflejar en este proyecto?
    Funcionalidad
    Modelado de Datos
    Implementacion de Middelwares
    Diseño Responsivo
    Implementacion de Estilos Personalizados utilizando CSS.
    Calidad de codificacion y documentacion con comentarios
    Uso de Javascript cont tecnicas de sincronismo.

Cómo pueden comenzar los usuarios con el proyecto?
    Al descargar/clonar el proyecto, deben tener instalado node.js y npm
    antes de ejecutarlo deben instalar todas los modulos con "npm install"
    ademas deben tener instaladado Mysql y crear una base de datos "db_myforum"
